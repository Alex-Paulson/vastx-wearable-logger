# Gui Plan

## Aim
Plan the future app screen for the Polar H10 wearable longer.

## Planned Screen Elements
- Participant ID box
- Session ID box
- Scan button
- Connect button
- Start recording button
- Stop recording button
- Save button
- Connection status
- Live value display
- Error/status messages
